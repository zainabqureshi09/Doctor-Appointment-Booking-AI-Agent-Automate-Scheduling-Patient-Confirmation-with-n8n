# Creating and Selling AI Agents for Appointment Automation

## 🤖 Introduction to AI Agent Earning Potential

The video explains how to create an AI agent that helps earn money by selling it as a monthly subscription or one-time sale.

The agent is designed primarily for professionals such as doctors to automate appointment bookings via WhatsApp.

The process uses a free tool called N8 (N8N) with detailed step-by-step instructions.

The video promises completion of the agent setup within 10 minutes and highlights its high market potential.

## 🛠️ Setting Up the AI Agent Workflow

Start by creating a free N8 account; a separate video is available for account setup.

Use the “Create Workflow” option within N8 to begin building the agent.

Add a WhatsApp Business node to trigger messages and connect it to the user’s WhatsApp credentials.

JavaScript code (provided in an accompanying PDF) is used to automate form generation and message handling.

Test the workflow by sending a message to confirm the agent responds correctly.

## 📅 Integrating Forms and Appointment Scheduling

Add an “On Form Submission” node to capture user inputs such as contact number, appointment timing, and doctor details.

Use Google Calendar API integration to automatically schedule appointments and send notifications.

Credentials for Google APIs are created through signing into Google and setting up the calendar event resource.

Drag-and-drop elements link form data to calendar events and WhatsApp triggers for seamless automation.

## 📧 Email and WhatsApp Notification System

Add Gmail node to send appointment confirmation emails to customers with a formatted text message.

Customize the email subject and body using data from the submission form.

Duplicate WhatsApp credentials to send notifications via WhatsApp alongside emails.

The automation ensures patients receive appointment details both on WhatsApp and email simultaneously.

## 💰 Monetizing Your AI Agent

Charge doctors a monthly subscription fee (₹500-₹600) for unlimited customer access to the AI agent.

The subscription fee covers API credentials and maintenance.

Even with 100 doctors subscribing, earnings can reach around ₹60,000 monthly.
